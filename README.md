# HotelRatingPrediction-regression_Model-EndtoEnd_deployment
Hotel Rating prediction using regression model and deployed in Streamlit


## Problem Statement: 
The main goal of this project is to perform extensive Exploratory Data Analysis(EDA) on the Zomato Dataset and build an appropriate Machine Learning Model that will help various Zomato Restaurants to predict their respective Ratings based on certain features

## Approach: 
The classical machine learning tasks like Data Exploration, Data Cleaning, Feature Engineering, Model Building and Model Testing. Used Regression algorithm for this dataset

## Dataset Link:
https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants?resource=download

## Evaluation metrics:
Accuracy            : 98%
r2 Score            : 0.93
Mean_Squared_Error  : 0.01
Mean_Absolute_Error : 0.06
